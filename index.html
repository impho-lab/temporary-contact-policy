2. Overview

Temporary Contact is an Android application that allows users to create temporary contacts that are automatically deleted after a user-defined period.

The application is designed with a local-first architecture. No data is transmitted to external servers.

3. Data we access

The application may access the following data on your device:

  • Contacts stored on your device (only when explicitly selected by the user)
  • Contact identifiers and display names

No sensitive personal data is collected beyond what is necessary for functionality.

4. Purpose of processing

Contact data is used exclusively for:

  • Creating temporary contacts
  • Scheduling automatic deletion of contacts
  • Displaying contact information inside the application

5. Legal basis (GDPR)

In accordance with the General Data Protection Regulation (GDPR), data processing is based on:

  • User consent when adding or managing contacts
  • Necessity for the execution of the application functionality

6. Data storage

All data is stored locally on the user's device using a local database. No cloud storage or external database is used.

Data is retained only until the user deletes it or the scheduled deletion is executed.

7. Automatic deletion

The application uses Android’s system scheduling service (AlarmManager) to automatically delete contacts at the scheduled time. This process runs entirely on the device.

8. Data sharing

No personal data is shared, sold, or transferred to third parties.

9. International data transfers

No data is transferred outside the European Union or to external servers.

10. User rights (GDPR)

Under the GDPR, you have the following rights:

  • Right of access to your data
  • Right to rectification
  • Right to erasure (“right to be forgotten”)
  • Right to restrict processing
  • Right to withdraw consent at any time

Since all data is stored locally on your device, these rights can be exercised directly within the application by deleting or modifying contacts.

11. Security

The application does not transmit data over the internet. Security depends on the Android operating system and device-level protections.

12. Third-party services

This application does not use analytics, advertising, tracking, or third-party SDKs.

13. Children’s privacy
